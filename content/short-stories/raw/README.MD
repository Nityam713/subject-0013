# raw/
*First drafts. Fragments. Things written on a whim at 2am that might be nothing or might be everything.*

---

## WHAT THIS FOLDER IS

This is where you write before you know what you are writing.

No structure required. No voice requirements. No bible compliance check.
You have an idea — a person, a moment, a line, a feeling — and you write it down
before it disappears. That is the only rule of this folder.

Raw content is **not a failure state.** It is the first state.
Every published story in this project started as something unclear in here.

---

## WHAT RAW CONTENT LOOKS LIKE

Raw drafts in this project tend to arrive in one of these forms:

**The Fragment** — a single scene, a single image, a single line that feels right
but has nothing around it yet. No beginning. No ending. Just the thing itself.
File it. It will find its context eventually.

**The Whim** — written fast, without stopping to check if it makes sense.
Contradictions allowed. Vague sections allowed. Missing quotes allowed.
The whim captures energy that editing cannot manufacture later.
Never edit a whim before filing it. File it as it came out.

**The Unclear Draft** — a full attempt at a story that doesn't quite work yet.
The subject isn't landing. The encounter feels off. The ending resolves when it shouldn't.
You know something is wrong but not what. File it anyway.
That is what `revised/` is for.

**The Note** — not even a draft. A character observation, a situation,
a "what if this person met 0013." Three sentences or three hundred words.
File it. Notes become fragments. Fragments become stories.

---

## HOW TO FILE HERE

Name your file descriptively. Not `draft1.md`. Not `untitled.md`.

Use the subject as the anchor:

```
[subject-descriptor]-[situation].md

examples:
  the-woman-who-stopped-reading.md
  man-on-the-train-platform.md
  archivist-session-fragment.md
  the-writer-first-encounter.md
```

If you don't know the subject yet, use the feeling:

```
  something-about-waiting.md
  the-line-about-blank-pages.md
  aftermath-no-subject-yet.md
```

---

## WHAT RAW DOES NOT NEED

- Bible compliance
- Correct 0013 voice
- A complete encounter structure
- Quotes that land
- An ending that works
- Consistency with other stories
- A title

All of these are `revised/`'s problem. Not yours. Not yet.

---

## THE ONE THING RAW MUST HAVE

The idea. The seed. The thing that made you want to write this.

Even if everything else is unclear — the subject is undefined,
the structure is missing, the prose is rough —
the original impulse must be recoverable from the file.

If you read it back and cannot find what you were reaching for,
add one line at the top: *"I was trying to write about \_\_\_."*
That is enough. Revised will take it from there.

---

## MOVING A FILE OUT OF RAW

A file leaves `raw/` when you are ready to hand it to the revision process —
either to yourself with fresh eyes, or to an AI with the brief below.

It does not need to be good. It needs to be *done being raw.*
There is a difference. Done being raw means: you have written everything
you know about this story in its current form. Nothing more is coming
without a different kind of attention.

That is when you move it to `revised/` and write the revision brief.

---

## THE REVISION BRIEF

When moving a file to `revised/`, attach a brief at the top of the file.
This is your instructions — to yourself or to an AI — about what this draft needs.

```
REVISION BRIEF
——————————————
What this story is trying to be:
[One or two sentences. What was the original impulse?]

What is working:
[What to preserve exactly. Lines, images, ideas that are right.]

What is not working:
[What is unclear, missing, or wrong. Be specific if you can.]

What it needs:
[Fill in missing quotes / Fix unclear sections / Enforce bible voice /
Add encounter structure / Other: ________]

Awareness level of subject:
[Level 0 — Unaware / Level 1 — Sensing / Level 2 — Knowing / Level 3 — Addressing]

0013's presence tier:
[Direct / Peripheral / Absent / Voiced]

Format type:
[Fragment / Document / Scenario / Confession / Other]
```

This brief is the handoff. Without it, revision is guessing.
With it, revision is surgery — precise, purposeful, preserving what matters.
