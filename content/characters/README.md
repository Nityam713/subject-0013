# Characters — how to file

## Tiers

| Tier | Folder | Use when |
|------|--------|----------|
| **1** | `tier-1/` | Recurring, narratively significant; can bend arc or voice |
| **2** | `tier-2/` | Multiple entries or pieces; not central but not disposable |
| **3** | `tier-3/` | Observed once, filed, moved on |

## File naming

Use kebab-case: `the-anomaly.md`, `woman-at-cafe.md`.

## Special case

`tier-1/the-anomaly.md` uses a **different structure** from other character files (define that structure inside that file when you write it).

Global foundation remains `../bible/0013-character-bible.md`.
